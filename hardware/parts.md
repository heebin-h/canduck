### RPi4 전환 영향 (2026-07-12 확정)

- **전원 설계 변경 없음**: RPi4 peak는 ~3A로 RPi5(5A)보다 낮음 → 5V 레일 마진만 커짐. power-budget.md 부하표는 RPi5 기준 보수치 그대로 둠.
- **HAT(Board B) 호환**: 40핀 헤더 위치·마운트홀(58×49mm)·보드 외형(85×56mm) 동일 → 풋프린트 수정 불필요. schematic 단계에서 커넥터 간섭만 육안 확인.
- **쿨링/PSU 교체**: RPi5 Active Cooler 미호환 → 방열판+팬(8K). PSU는 27W PD → 15W USB-C(보유 시 생략).
- **성능**: v1 호스트 부하(Porcupine wake-word + FSM + MQTT + 텔레메트리)는 RPi4로 충분. v2 백로그의 로컬 LLM은 RPi4에선 비현실적 → 클라우드 API 경로 사용.

## 발주 확정 (2026-07-12)

### 실장 방식 — 혼합

- **Board A (Power)**: JLCPCB SMT 어셈블리. 파워계 IC(벅×2, LDO, INA219×2, 션트, MOSFET, 퓨즈, 제너, NTC)와 Board A용 R/C는 **지금 사지 않고** W2 PCB 발주 시 JLC Parts Library에서 통합 발주. schematic 부품 선정 시 JLC 라이브러리 재고 부품 우선.
- **Board B (HAT)**: 직접 실장 (핫에어+페이스트). Board B 실장 IC/부품은 배치 1에서 지금 발주.

## 대체품 / 다이어트 옵션

- RPi 5 8GB → 4GB (-30,000): LLM 로컬 안 돌릴 거면 4GB 충분
- 오실로스코프 → 로직 애널라이저만 (-130,000): 디지털 신호만 본다면. 아날로그 디버깅 필요해지면 후회.
- USB 현미경 → 헤드밴드 루페 (-85,000): SMT 0805 이상이면 루페로 충분
- 외주 출력 → 본인이 3D 프린터 구매 (Bambu A1 mini 350K): 2회 이상 출력 + 미래 프로젝트 ROI 있음
