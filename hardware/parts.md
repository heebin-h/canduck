### RPi4 전환 영향 (2026-07-12 확정)

- **전원 설계 변경 없음**: RPi4 peak는 ~3A로 RPi5(5A)보다 낮음 → 5V 레일 마진만 커짐. power-budget.md 부하표는 RPi5 기준 보수치 그대로 둠.
- **HAT(Board B) 호환**: 40핀 헤더 위치·마운트홀(58×49mm)·보드 외형(85×56mm) 동일 → 풋프린트 수정 불필요. schematic 단계에서 커넥터 간섭만 육안 확인.
- **쿨링/PSU 교체**: RPi5 Active Cooler 미호환 → 방열판+팬(8K). PSU는 27W PD → 15W USB-C(보유 시 생략).
- **성능**: v1 호스트 부하(Porcupine wake-word + FSM + MQTT + 텔레메트리)는 RPi4로 충분. v2 백로그의 로컬 LLM은 RPi4에선 비현실적 → 클라우드 API 경로 사용.

## 발주 확정 (2026-07-12)

### 실장 방식 — 전량 JLCPCB SMT 어셈블리 (2026-07-12 최소화로 변경, 기존 "혼합" 대체)

- **Board A + Board B 둘 다 JLC 어셈블리.** 보드 실장 부품(ESP32-S3 모듈, PCA9685, MPR121, MPU6050, TCA9548A, CH340N, PAM8403, TB6612FNG, INA219×2, 션트, LDO, MOSFET, 퓨즈, 제너, NTC, USB-C 커넥터, 택트버튼, R/C)은 **지금 아무것도 사지 않고** W2 PCB 발주 시 JLC Parts Library에서 통합.
- 변경 근거: 직접 실장을 빼면 핫에어/페이스트/현미경급 장비(-112K+)와 실장 실패 리스크가 통째로 사라짐. 어셈블리 수수료 증가분(+3~5만 추정)보다 절감이 큼.
- 대가: schematic 부품 선정을 **JLC 라이브러리 재고 부품으로 맞춰야 함** (W1 설계 시 제약). Extended part는 수수료 있으니 Basic part 우선.
- 예외(직접 납땜, 인두로 충분): Pololu 벅 모듈×2(스루홀 헤더), 커넥터/헤더류, 패치 와이어.

- **AvantGarde 또는 Mouser**: 정확한 datasheet 매칭 필요한 IC (INA219, PCA9685 등)
